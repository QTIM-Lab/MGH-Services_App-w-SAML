Support

Getting Started:

If you need help or would like to volunteer, please review existing posts. If you find a post that matches a service you need or one that you would like to offer, please email the poster directly.

If you do NOT see a post that matches your request or offer, click Post a New Request for one of the following types of services:

  • Transportation
  • Shopping
  • In-House Help
  • Pet Sitting/Care

Please note that this site should not be used to request coverage of a hospital shift or to arrange for child care. If you need child care support, please click here.

This Site Can Also Help You

  • Search the “Post” field for key words or phrases to help you find exactly what you are looking for
  • Sort search ressults
  • Update an existing post, to either "Resolved" or "I don't ned help anymore"

Tips:

  • If you mark a post "Resolved" or "I don’t need help anymore" you will not be able to retrieve it.
  • Use the map to look for people nearby who can help/need help.
    • Click on an icon to see more detail.
    • Hit refresh to see your request appear on the map
    • If you choose to help vs receive help, your icon shows up as a red heart.
  • If posts and or icons aren't showing up, try hitting 'ctrl+shift+r' or doing a hard refresh.
